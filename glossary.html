The Many Folders Of Linux!

When using the Command Line, you will need to know the different directories available. Here is a breakdown of the directories available in most Linux distros!

Linux File Directories
Directory Description
/ The root directory, the top-level directory of the Linux file system hierarchy.
/home Contains personal directories for each user, where individual files, configurations, and data are stored.
/etc Stores system-wide configuration files and scripts used for system setup and management.
/var Holds variable files like logs (/var/log), cache, and temporary data that frequently changes during runtime.
/usr Contains user application files and libraries, including shared resources for installed software.
/bin Houses essential binaries (commands) required for the system to operate in single-user mode.
/sbin Contains system binaries used for administrative tasks, like fsck, reboot, and ifconfig.
/lib Stores essential shared libraries used by the binaries in /bin and /sbin.
/tmp Temporary storage directory for files created by programs; data here is cleared upon system reboot.
/opt Optional software directory for third-party applications and additional packages.
/dev Contains device files that provide access to hardware such as hard drives, printers, and keyboards.
/boot Holds files needed for the booting process, including the Linux kernel and initial ramdisk.
/proc A virtual directory providing information about currently running processes and kernel parameters.
/sys Provides detailed system information and exposes kernel objects for interaction.
/media Used for automatically mounting removable devices like USB drives and CDs.
/mnt A generic mount point for temporarily mounting filesystems.
/root The home directory for the root (administrative) user.
/run Stores system information and runtime data during the current boot session.
/lost+found Contains recovered files after a filesystem check (used for troubleshooting).

Commanding The Console

There are many useful commands when using the Command Line in Linux. Here are some useful commands to help you get the most out of your machine!

Linux Commands
Command Explanation
ls Lists files and directories in the current working directory.
cd Changes the current directory to the specified path.
pwd Prints the absolute path of the current working directory.
cp Copies files or directories from one location to another.
mv Moves (or renames) files and directories.
rm Removes files or directories (use carefullyâ€”there's no undo!).
mkdir Creates a new directory with the specified name.
touch Creates an empty file or updates the timestamp of an existing file.
cat Concatenates and displays the contents of files.
nano Opens the Nano text editor to edit files directly in the terminal.

Distributed Solutions

There are many different Linux distros available and it can be difficult to pick the right one to suit your needs. Here is a breakdown on some of the most popular distros and why you should use them!

Popular Linux Distributions
Distribution Description
Ubuntu A user-friendly distro that's ideal for beginners, with extensive community support and a focus on ease of use.
Debian A stable and versatile distro used as the foundation for many others, including Ubuntu.
Fedora Focuses on innovation and includes the latest open-source technologies. Often chosen by developers.
Arch Linux A lightweight and highly customizable distro for advanced users who prefer full control over their system.
Linux Mint Based on Ubuntu/Debian, Mint is designed to be simple, elegant, and perfect for those switching from Windows.
CentOS A free, enterprise-grade distribution based on Red Hat Enterprise Linux (RHEL), suitable for servers.
Kali Linux A security-focused distro equipped with tools for penetration testing and ethical hacking.
elementary OS An Ubuntu-based distro with a macOS-like user interface, prioritizing design and user experience.
Manjaro A beginner-friendly spin on Arch Linux, combining simplicity with rolling-release updates.
Zorin OS A distro designed to feel familiar to Windows and macOS users, making it a great choice for newcomers.
